# Installation Guide

## Prerequisites

- Node.js 18+
- npm or yarn
- Git

## Step 1: Clone Repository

```bash
git clone https://github.com/romero429-collab/Kiyoshi-AI-OS.git
cd Kiyoshi-AI-OS
```

## Step 2: Install Dependencies

```bash
npm install
```

## Step 3: Build Project

```bash
npm run build
```

## Step 4: Run System

```bash
npm start
```

## Troubleshooting

- **Node not found**: Install from nodejs.org
- **Build fails**: Run `npm install` again
- **Permission denied**: Use `sudo` or check permissions
